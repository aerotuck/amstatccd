Introduction

In this post, we will install blogdown in R, install Hugo, and install a Hugo theme. Note that you need to have R installed to follow along. Any editor will do.

Installation

In order to install and use blogdown, just enter the following code in the R terminal, or from within a script.

install.packages("blogdown", repos = "https://cloud.r-project.org/")
library(blogdown)

Next, to install Hugo, simply enter the following:

blogdown::install_hugo(force=TRUE)
## The latest Hugo version is v0.88.1
## Hugo has been installed to "/root/.local/share/Hugo/0.88.1". You are recommended to set options(blogdown.hugo.version = "0.88.1") in the .Rprofile file in your website project. See the blogdown book for more info on .Rprofile: https://bookdown.org/yihui/blogdown/global-options.html

Installing themes

Hugo has a huge number of pre-configured themes. We note, you should select a Hugo theme that supports blogging, as there are a variety of themes that serve various functions. There are many, many themes that are not included in that set of themes linked above. This site uses hugo-lithium. To install hugo-lithium, enter the following:

blogdown::install_theme(theme = 'yihui/hugo-lithium', force=TRUE)
## Warning: The theme has provided an example site. You should read the theme's
## documentation and at least take a look at the config file config.toml (or .yaml)
## of the example site, because not all Hugo themes work with any config files.

That’s it! You’ve now installed Blogdown, Hugo and a theme on your computer. In the next post I will describe how to connect your site to Netlify.
